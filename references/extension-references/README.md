# Extension References

Local-only reference folders used while developing Piko.

Do not commit these folders into the Piko repository. They are third-party unpacked Chrome extensions used only for architecture, UX, and implementation study.

## Folder Map

- `instagram/turbo-downloader-instagram`
  - Reference for Instagram media detection, page-context bridge patterns, route observation, and downloader survivability.
- `youtube/viewstats-youtube-analytics`
  - Reference for contextual YouTube overlays, SPA route awareness, and lightweight inline/floating controls.
- `youtube/ostendo-youtube-screenshots`
  - Reference for polished YouTube actions, compact interaction surfaces, and motion/feedback quality.
- `workflow-ui/clickpilot-clipper`
  - Reference for restrained contextual controls and compact workflow UI.

## Rules

- Study behavior and architecture, not source code to copy.
- Keep notes in the Piko repo using original summaries such as `REFERENCE_NOTES.md`.
- Keep this folder outside Git.
- When sharing context with cloud agents, share summaries/screenshots, not raw third-party extension files.
